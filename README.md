# Trading Platform

> A sleek, data-driven **online trading platform** UI for stocks, crypto, and commodities — featuring real-time charts, portfolio tracking, and market analysis tools.

🌐 **Live Demo:** [Trading-Platform on GitHub Pages](https://pinnacleroute.github.io/Trading-Platform)

---

## ✨ Features

- 📈 Real-time market charts & indicators
- 💼 Portfolio overview & asset allocation
- 🔔 Price alerts & watchlists
- 📉 Profit/loss tracker
- 📊 Market news & sentiment feed
- 📱 Fully responsive UI

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | HTML5, CSS3, JavaScript |
| Deployment | GitHub Pages |
